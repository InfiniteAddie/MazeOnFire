Commands
q | quit: quit the program
g: generate new maze
p | print: print maze
r: a reference for values printed by 'p' or 'print'
pr: print readable version of maze (creates output.txt)
dfs: run DFS on maze
bfs: run BFS on maze
a: run A* on maze
p[dfs | bfs | a]: runs respective search algorithm on maze and prints result
pr[dfs | bfs | a]: runs respective search algorithm on maze and prints readable result (creates output.txt)
strat[1 | 2 | 3]: run strategy 1, 2, or 3 on the maze
h | help: show this list again
h [g | p | print | pr] | help [g | p | print | pr]: get more info on command
